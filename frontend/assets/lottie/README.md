# Lottie Animations

This directory is intended to store Lottie animation JSON files used in the application.

Please download the following animations from LottieFiles and save them in this directory with the specified filenames. The application's animation components expect these files to be present with these exact names.

1.  **Loading Spinner**
    *   Filename: `loading_spinner.json`
    *   Download URL: [https://lottiefiles.com/animations/loading-animation-N7CMywV2k3](https://lottiefiles.com/animations/loading-animation-N7CMywV2k3)

2.  **Success Tick**
    *   Filename: `success_tick.json`
    *   Download URL: [https://lottiefiles.com/animations/success-tick-03VosS8H9P](https://lottiefiles.com/animations/success-tick-03VosS8H9P)

3.  **Error Cross**
    *   Filename: `error_cross.json`
    *   Download URL: [https://lottiefiles.com/animations/error-NQxINq1p6n](https://lottiefiles.com/animations/error-NQxINq1p6n)

4.  **Trading Arrows**
    *   Filename: `trading_arrows.json`
    *   Download URL: [https://lottiefiles.com/animations/arrows-Vz9F9Z8vSg](https://lottiefiles.com/animations/arrows-Vz9F9Z8vSg)

**Instructions:**
- Click on each URL.
- On the LottieFiles page, find the "Download" button (it might be under a kebab menu or share icon).
- Select the option to download the "Lottie JSON" (usually a `.json` file).
- Save the file into this `frontend/assets/lottie/` directory, making sure the filename exactly matches the one specified above.
